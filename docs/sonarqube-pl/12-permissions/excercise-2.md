# Nadanie uprawnień projektowych

## Cel 
 
Zrozum, jak nadawać uprawnienia specyficzne dla projektu.

## Zadania

1. Przejdź do **Administration > Security > Permissions** dla konkretnego
   projektu.
2. Nadaj uprawnienia projektowe, np. **Administer Project** albo
   **Execute Analysis**, użytkownikowi lub grupie.
3. Zweryfikuj, że użytkownik/grupa może wykonywać dozwolone działania
   w projekcie.
