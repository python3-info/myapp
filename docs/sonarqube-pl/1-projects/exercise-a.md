# Nawigacja po liście projektów

## Cel  

Naucz się nawigować po liście projektów i zidentyfikować
kluczowe informacje o każdym projekcie.

## Zadanie

1. Otwórz instancję SonarQube.
2. Z menu u góry wybierz **Projects**.
3. Dla projektu zidentyfikuj następujące informacje:

   - **Project Name**
   - **Quality Gate Status** (np. Passed/Failed)
   - **Last Analysis Date**
