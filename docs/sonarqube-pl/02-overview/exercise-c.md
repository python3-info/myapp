# Activity projektu

## Cel

Przeglądaj ostatnią aktywność projektu, aby zrozumieć, 
jak zmieniała się jakość kodu w czasie.

## Zadania

1. Przejdź do sekcji **Activity**.
2. Przeglądaj ostatnie zmiany w projekcie, takie jak push kodu,
   wyniki analizy czy rozwiązywanie issue.
3. Zidentyfikuj wzorce i trendy w jakości kodu.
