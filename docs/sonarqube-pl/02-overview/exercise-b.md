# Nawigacja po stronie Project Overview

## Cel

Poznaj układ strony i najważniejsze informacje na stronie Project Overview.

## Zadania

1. Otwórz stronę **Project Overview** dla wybranego projektu.
2. Wskaż miejsce, gdzie jest informacja o wyniku sprawdzania **Quality Gate**.
3. Zidentyfikuj **kluczowe metryki**:

   - **Security**
   - **Reliability**
   - **Maintainability**
   - **Hotspots Reviewed**
   - **Coverage**
   - **Duplications**
