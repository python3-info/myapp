# Filtrowanie projektów po metrykach jakości

## Cel  

Naucz się filtrować projekty na podstawie kluczowych metryk jakości.

## Zadanie

1. Otwórz instancję SonarQube.
2. Z menu u góry wybierz **Projects**.
3. Przejrzyj sekcję **Filters** widoczną po lewej stronie.
4. Przefiltruj projekty po następujących metrykach:

   - **Quality Gate** (np. Passed)
   - **Security** (np. poziom C lub mniejszy)
   - **Coverage** (np. poniżej 80%)
   - **Duplications** (np. powyżej 10%)
